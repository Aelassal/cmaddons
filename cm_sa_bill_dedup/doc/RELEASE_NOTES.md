# Release Notes

All notable changes to **Duplicate Vendor Bill Warner** are tracked here.

## 19.0.1.0.0 — 2026-04-24

Initial public release on the Odoo App Store.

**What's in it**

* Fuzzy-match incoming vendor bills against history to flag near-duplicates.
* Compatible with Odoo 17.0, 18.0, and 19.0 (Community + Enterprise).
* No core-model inheritance, safe to install and uninstall.
* Full user manual shipped in `docs/user_manual.pdf`.

**Known limitations**

None at release. If you hit anything that looks like a bug, email
<contact@cm.sa> and we will ship a fix.
